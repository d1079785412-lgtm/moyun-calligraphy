这是“墨韵智创”静态演示版。

部署方式：
1. 打开 https://app.netlify.com/drop
2. 把整个 static-demo 文件夹拖进去
3. 等几秒，复制生成的 https://xxx.netlify.app 链接

如果使用 Cloudflare Pages：
1. 打开 Cloudflare Pages
2. 选择 Direct Upload
3. 上传 static-demo 文件夹

说明：
- 这个版本不需要服务器，也不需要 Vercel。
- 问答、章法建议、图片生成都是浏览器本地 mock。
- 历史作品保存在当前浏览器 localStorage。
